# Reproduction de DiceHuBERT

Cette expérimentation vise à reproduire **DiceHuBERT**. DiceHuBERT est un modèle auto-supervisé distillé à partir de *HuBERT_base*.

Les articles associés :
- [HuBERT](arxiv.org/abs/2106.07447)
- [DiceHuBERT](https://arxiv.org/abs/2507.02911)

## Modèles
Le modèle **DiceHuBERT** est l'équivalent du modèle **HuBERT-Narrow** qui est le modèle **HuBERT** avec la taille des dimensions de ses représentations divisées par deux
